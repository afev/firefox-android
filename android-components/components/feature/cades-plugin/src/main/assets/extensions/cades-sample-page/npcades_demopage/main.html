Компания КРИПТО-ПРО предлагает плагин для создания и проверки электронной подписи на web-страницах.

КриптоПро ЭЦП Browser plug-in позволяет создавать и проверять как обычную электронную подпись, так и усовершенствованную электронную подпись. Подробнее на нашем сайте.


Обратите внимание! Для пробной работы с плагином вам необходимо иметь:
  • Компьютер под управлением Windows, Linux или MacOS
  • Один из современных браузеров (Internet Explorer, Mozilla Firefox, Opera, Chrome, Яндекс.Браузер, Safari, Chromium Edge) с поддержкой сценариев JavaScript
  • Установленный плагин для браузера «КриптоПро ЭЦП Browser plug-in» (Установить)
  • Если планируется создание ЭЦП по ГОСТ Р 34.10-2001/2012, то необходимо установить СКЗИ КриптоПро CSP
  • Сертификат ключа подписи, который можно получить на странице тестового центра
